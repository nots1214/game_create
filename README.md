# game_create

Tetris:2022年頃に作成したコード。基本的なシステムが実装されていて、通常通り遊ぶことができる。

NewTetris:2026年に改良したTetrisのコード。色の追加、長押しによる速い移動、その他細かいところを修正。また、オブジェクト指向による設計に修正。

3/22追記
UIを全面的に変更。
更に新モードを追加し、盤面を回転させることが可能に。盤面を回転後に上手くテトリミノをはめると特殊技ができるようにした。
チャレンジモードは上級者向けではあるが、回転技の練習にすることもできる。

できること:PERFECT CLEAR(全消し)、連鎖(REN)、回転入れ(TspinやIspinなど)、縦消し(縦に１列そろえると消える)、Bspin(盤面を回転させ、今降っているミノに盤面を綺麗にはめると発動)

操作方法:A左移動 D右移動 Wハードドロップ Sソフトドロップ O左回転 P右回転 Iホールド L盤面を左に回転　R盤面を右に回転　J盤面を180度回転　Esc終了
